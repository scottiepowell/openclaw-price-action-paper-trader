# Alpaca Paper Adapter Plan

This is a future-phase plan only.

## Requirements before implementation

- Strategy Lab import works
- Order-plan generation works
- Risk gate works
- File journal works
- Manual approval workflow exists
- Paper mode config exists
- Live trading remains unsupported

## Adapter constraints

- Paper account only
- No live account support
- No default order submission
- All broker responses must be journaled
- All submissions require explicit approval
